Share Structure

Two classes of shares. Same economics. Different governance rights.

Class A (Voting)

  • Vote on investments >$15k
  • 24-hour voting period
  • 30% quorum required
  • 51% approval needed
  • Same returns as Class B
  • For GPs + strategic LPs

Class B (Non-Voting)

  • Passive participation
  • No governance overhead
  • Same returns as Class A
  • Redeem anytime (NAV)
  • For capital-only LPs
  • Simple & efficient
